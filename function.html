1. Introduction to Functions

Functions are modular blocks of code designed to perform a specific task. They help reduce code redundancy and improve program structure.

2. Need for Functions

  • Modularization of code
  • Reusability
  • Improved readability and maintainability

3. Types of Functions

  • Library Functions (e.g., printf(), scanf())
  • User-Defined Functions

4. Structure of a Function

ReturnType FunctionName(Parameter List)
{
    // Body of the function
}

Example:
int sum(int a, int b) { return a + b; }

5. Function Call and Declaration

  • Function Declaration: tells the compiler about the function's name and signature.
  • Function Call: invokes the function.
  • Function Definition: the actual body of the function.

6. Categories of Functions

  • No arguments, no return value
  • Arguments, no return value
  • No arguments, return value
  • Arguments and return value

7. Example

#include <stdio.h>

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(5, 3);
    printf("Sum: %d", result);
    return 0;
}
