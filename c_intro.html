What is C?

C is a general-purpose, procedural programming language developed in the early 1970s by Dennis Ritchie at Bell Labs. It has influenced many other languages like C++, Java, and Python.

Why Learn C?

  • It forms the foundation of many modern languages.
  • Offers fine-grained control over system resources.
  • Ideal for system-level programming like OS and compilers.

Basic Structure of a C Program

#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}

Every C program must have a main() function as its entry point.

Key Features

  • Fast and efficient
  • Supports pointers and direct memory access
  • Portable across different platforms
